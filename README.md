# 🍉 FruitMerge Game

Popüler Japon oyunu **Suika Game (Karpuz Oyunu)** konseptinden ilham alınarak **Unity** ve **C#** ile geliştirilmiş, eğlenceli ve bağımlılık yapan bir meyve birleştirme oyunu!

---

## 🎮 Hakkında

FruitMerge Game, fizik tabanlı bir bulmaca oyunudur. Oyuncular meyveleri bir kaba bırakır ve aynı türdeki iki meyve çarpıştığında birleşerek bir üst seviyedeki meyveye dönüşür. Amaç, kabı taşırmadan mümkün olan en yüksek skoru elde etmektir.

## 🕹️ Nasıl Oynanır?

1. **Meyveyi bırak** — Tıklayarak veya dokunarak meyveyi kaba bırakın.
2. **Birleştir** — Aynı türdeki iki meyve çarpıştığında, bir üst seviyedeki meyveye dönüşür.
3. **Stratejik düşün** — Meyveler gerçekçi fizik kurallarına göre hareket eder, bırakışlarınızı iyi planlayın.
4. **Taşırma!** — Meyveler tehlike çizgisini aşarsa oyun biter.
5. **Puan kazan** — Her birleştirmede puan kazanırsınız. Büyük meyveler daha çok puan getirir!

## 🔧 Kullanılan Teknolojiler

- **Motor:** Unity
- **Programlama Dili:** C# (%100)
- **Fizik:** Unity 2D Physics (Rigidbody2D, CircleCollider2D)

## 🚀 Başlangıç

### Gereksinimler

- Sisteminizde **Unity Hub** kurulu olmalıdır.
- **Unity Editor** (kullanılan sürüm için `ProjectSettings/ProjectVersion.txt` dosyasını kontrol edin)

### Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/0Alduin0/FruitMerge-Game.git
   ```
2. **Unity Hub**'ı açın ve **"Open"** butonuna tıklayarak klonlanan proje klasörünü seçin.
3. Unity'nin tüm varlıkları içe aktarmasını bekleyin.
4. `Assets/Scenes` klasöründen ana sahneyi açın ve **Play** butonuna basın.
